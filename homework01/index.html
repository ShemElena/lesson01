git init - создание нового локального репозитория

git status - состояние репозитория

git add - добавление файлов к коммиту

git commit - создание коммита

git log - история коммитов

git checkout - переключение между ветками

git branch - создание ветки

git push - перемещает коммиты из локального в удаленный репозиторий

git pull - извлечение и загрузка содержимого из удаленного репозитория и обновления локального репозитория этим содержимым

git merge - добавление новых наработок в ветку

git clone - копирование удалённого репозитория на локальный ПК

.gitignore - специальный файл, в который заносятся файлы, которые не должны попадать в коммиты

  • git revert - отменяет внесённые изменения в коммите
  • git reset - отменяет локальные изменения в репозитории Git
  • git rebase - объединяет нескольких коммитов в новый базовый коммит, не использовать в публичных коммитах
  • git cherry-pick - выбирает коммит из одной ветки и применяет его к другой
  • git stash - позволяет временно отложить изменения
